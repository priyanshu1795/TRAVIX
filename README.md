TRAVIX – AI Trip Planner

TRAVIX is a smart and interactive AI-based travel planning web application developed using HTML, CSS, and JavaScript. The project helps users generate personalized and budget-friendly travel plans based on their preferences such as destination, budget, travel type, and trip duration. The platform is designed with a modern UI, smooth animations, and dynamic recommendation features to provide users with an engaging travel planning experience.

Smart AI Trip Suggestions
Generates personalized travel plans dynamically
Suggests destinations according to user budget

Creates day-wise itinerary plans

How the Project Works
User enters trip details such as:
Destination
Budget
Number of days
Travel type
Season
JavaScript processes the inputs using predefined travel data and logic.
The system dynamically generates:
Suggested destinations
Estimated trip budget
Hotels and attractions
Day-wise itinerary
Travel and food recommendations
The generated result is displayed in an interactive and visually attractive format.
